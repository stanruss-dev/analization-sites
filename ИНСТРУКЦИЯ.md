# Site Analyzer - Инструкция по использованию

Программа заходит на любой сайт, обходит все страницы и собирает товары с ценами, контакты, описание компании, услуги и весь текстовый контент. Результат - структурированный Markdown-отчёт и база данных SQLite.

---

## Установка (один раз)

**1. Проверить Python** (нужен 3.10+):
```
python --version
```

**2. Перейти в папку программы:**
```
cd "E:\Aiprojects\Analize-saits"
```

**3. Установить зависимости:**
```
pip install requests beautifulsoup4 lxml playwright
```

**4. Установить браузер:**
```
python -m playwright install chromium
```

---

## Запуск

Открыть PowerShell (`Win+R` → `powershell`) и перейти в папку:
```
cd "E:\Aiprojects\Analize-saits"
```

### Базовый запуск
```
python scraper.py https://example.com
```

### С сохранением JSON
```
python scraper.py https://example.com --json report.json
```

### Полный обход большого сайта
```
python scraper.py https://example.com --max-pages 200 --json report.json
```

### Посмотреть отчёт без повторного обхода
```
python scraper.py https://example.com --report-only
```

---

## Параметры

| Параметр | Описание | По умолчанию |
|---|---|---|
| `https://...` | Адрес сайта (обязательно) | - |
| `--max-pages 200` | Сколько страниц обойти | 100 |
| `--workers 5` | Параллельных потоков | 5 |
| `--wait 800` | Ожидание JS в миллисекундах | 800 |
| `--json report.json` | Сохранить результат в JSON | нет |
| `--report-only` | Только показать отчёт из БД | нет |

---

## Примеры

**Небольшой сайт, быстро:**
```
python scraper.py https://example.com --max-pages 30
```

**Интернет-магазин, полный обход:**
```
python scraper.py https://shop.ru --max-pages 300 --workers 5 --json report.json
```

**Медленный или JS-тяжёлый сайт:**
```
python scraper.py https://site.ru --max-pages 50 --wait 2000 --workers 3
```

**Просмотр уже собранных данных:**
```
python scraper.py https://example.com --report-only
```

---

## Результаты

После запуска в папке появятся файлы:

- **`report_имя-сайта.md`** - главный отчёт: описание компании, все товары с ценами, контакты, содержимое страниц. Открывается в любом Markdown-редакторе (VS Code, Obsidian, Typora).
- **`site_data.db`** - база данных SQLite. Открывается программой [DB Browser for SQLite](https://sqlitebrowser.org). Таблицы: `pages`, `products`, `contacts`, `services`, `links`.
- **`report.json`** - полный отчёт в JSON (если указан `--json`). Импортируется в Excel или Power BI.
- **`logs/app.log`** - лог работы. Если что-то пошло не так - смотреть сюда.

## Частые вопросы

**Программа застряла или работает очень долго**
Нажать `Ctrl+C` для остановки. Уменьшить `--max-pages` или увеличить `--wait`.

**Мало товаров или не нашло цены**
Сайт может использовать нестандартную структуру. Попробовать `--wait 2000` чтобы дать больше времени на загрузку JavaScript.

**Ошибка про playwright**
Выполнить: `python -m playwright install chromium`

**Как запустить повторно на том же сайте**
Просто запустить снова - старые данные перезапишутся. Если нужно сохранить предыдущие результаты, переименовать `site_data.db` перед новым запуском.
